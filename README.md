# Automatic Visitor Counter using Arduino

## Overview
This project counts the number of visitors entering and exiting a room using an ultrasonic sensor. The count is displayed on a 16x2 LCD.

## Components
- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- 16x2 LCD with I2C
- Jumper Wires

## Features
- Counts entries
- Counts exits
- Displays total visitors
- Real-time LCD display
